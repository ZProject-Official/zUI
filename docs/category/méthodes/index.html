Méthodes

📄️ GoBack

La méthode GoBack est utile lorsque vous souhaitez revenir à un menu parent dans l'interface utilisateur.

📄️ Goto

La méthode Goto est utilisée pour naviguer vers un menu spécifique dans l'interface utilisateur.

📄️ IsVisible

La méthode IsVisible est utilisée pour vérifier si un menu principal est actuellement visible.

📄️ OnClose

La méthode OnClose est utilisée pour définir une fonction à exécuter lors de la fermeture d'un menu principal.

📄️ OnOpen

La méthode OnOpen est utilisée pour définir une fonction à exécuter lors de l'ouverture d'un menu principal.

📄️ SetItems

La méthode SetItems est utilisée pour définir les éléments d'un menu.

📄️ SetVisible

La méthode SetVisible est utilisée pour afficher ou masquer un menu principal.
