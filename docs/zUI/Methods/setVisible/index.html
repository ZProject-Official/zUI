SetVisible

La méthode SetVisible est utilisée pour afficher ou masquer un menu principal.

Fonction SetVisible​

La syntaxe pour utiliser la méthode de visibilité est la suivante :

zUI:SetVisible(IsVisible)

Cette méthode permet de gérer l'état de visibilité du menu, en affichant ou cachant l'interface correspondante selon le paramètre IsVisible.

Paramètres​

  • IsVisible [boolean] : Un booléen indiquant si le menu doit être visible (true) ou caché (false).

Exemple d'utilisation​

Voici un exemple d'utilisation de la méthode SetVisible :

    local Menu = zUI.CreateMenu("Mon Menu", "Sous-Titre", "https://example.com/banner.png", nil , "Description de mon menu")
    --Ici définir les items
    RegisterCommand("exemple", function()
        Menu:setVisible(not Menu:IsVisible())
    end, false)

Dans cet exemple, vous pouvez ouvrir le menu via une commande au lieu d'une touche

Résumé​

Pour utiliser la méthode SetVisible dans zUI :

Utilisez zUI:SetVisible(IsVisible) pour gérer l'affichage du menu.
